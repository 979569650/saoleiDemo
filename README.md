# 扫雷游戏

一个基于Vue3 + TypeScript + Tailwind CSS的经典扫雷游戏实现。

## 特性

- 🎯 经典扫雷玩法
- 📱 响应式设计
- 🎨 使用SVG图标，无外部图片依赖
- ⚡ 支持三种难度：初级、中级、高级
- ⏱️ 实时计时器
- 🚩 右键标记地雷功能
- 🎮 左键揭开方格，自动展开空白区域

## 技术栈

- Vue 3.4
- TypeScript 5
- Tailwind CSS 3
- Vite 5

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 类型检查
npm run type-check
```

## 游戏规则

1. 左键点击方格揭开
2. 右键点击方格标记/取消标记地雷
3. 数字表示周围8个方格中地雷的数量
4. 揭开所有非地雷方格即可获胜
5. 点击到地雷则游戏失败

## 难度设置

- **初级**: 9×9 方格，10个地雷
- **中级**: 16×16 方格，40个地雷  
- **高级**: 16×30 方格，99个地雷
